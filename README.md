
## Getting Started

Create .env file, copy the value from .env.example

First, run the development server:

```bash
npm install
#then 
npm install file:./ckeditor5
#then
npx prisma db push 
#then 
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
